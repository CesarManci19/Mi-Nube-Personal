# Mi Lista de Tareas

Este proyecto es una aplicación simple hecha con React y Vite.  
Sirve para agregar tareas a una lista y también eliminarlas cuando ya no se necesitan.

## ¿Qué hace la aplicación?

- Permite escribir una tarea en un cuadro de texto.
- Agrega la tarea a una lista.
- Permite borrar tareas con un botón.
- Muestra un diseño simple y ordenado.

## Tecnologías usadas

- React
- Vite
- JavaScript
- CSS

## Estructura del proyecto

- `App.jsx`: archivo principal de la aplicación.
- `App.css`: estilos de la aplicación.
- `TaskForm.jsx`: componente para escribir y agregar tareas.
- `TaskList.jsx`: componente para mostrar y eliminar tareas.

## ¿Cómo instalarlo?

1. Clona el proyecto.
2. Entra a la carpeta del proyecto.
3. Ejecuta este comando:

```bash
npm install
```

## ¿Cómo ejecutarlo?

Para iniciar la aplicación, usa:

```bash
npm run dev
```

Después abre el enlace que aparece en la terminal, normalmente es:

```bash
http://localhost:5173/
```

## ¿Cómo se usa?

1. Escribe una tarea en el campo de texto.
2. Presiona el botón **Agregar**.
3. La tarea aparecerá en la lista.
4. Si quieres eliminarla, presiona **Eliminar**.
